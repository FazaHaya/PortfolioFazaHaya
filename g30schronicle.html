G30SCHRONICLE

Aplikasi edukasi interaktif seputar sejarah G30S/PKI dengan fitur sejarah, video dokumenter, kuis interaktif, tanya AI, dan chat box antar pengguna.

📜 Latar Belakang

G30SCHRONICLE dikembangkan untuk memperkenalkan sejarah G30S/PKI dengan pendekatan digital yang menarik dan interaktif. Aplikasi ini menjadi sarana edukasi yang relevan untuk generasi muda agar tidak melupakan sejarah kelam bangsa.

🔍 Fitur Unggulan

  • 📖 Halaman Sejarah Lengkap G30S/PKI
  • 🎥 Video Dokumenter dan Narasi
  • 🧠 Kuis Interaktif Seputar Materi
  • 💬 Chat Box Edukasi untuk Diskusi
  • 🤖 Fitur Tanya AI terkait G30S/PKI

🛠️ Teknologi yang Digunakan

  • 🌐 Laravel + Blade
  • 🎨 Tailwind CSS
  • 🤖 OpenAI API untuk Tanya AI
  • 🗄️ MySQL
